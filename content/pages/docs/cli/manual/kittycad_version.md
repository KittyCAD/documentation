Prints the version of the program.

### Options

<dl class="flags">
   <dt><code>-w/--web</code></dt>
   <dd>Open the version in the browser</dd>

   <dt><code>-h/--help</code></dt>
   <dd>Print help information</dd>

   <dt><code>-d/--debug</code></dt>
   <dd>Print debug info</dd>
</dl>
