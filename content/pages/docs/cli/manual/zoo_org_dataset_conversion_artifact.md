Write a conversion artifact.

### Options

<dl class="flags">
   <dt><code>dataset-id</code></dt>
   <dd>Dataset ID</dd>

   <dt><code>conversion-id</code></dt>
   <dd>Conversion ID</dd>

   <dt><code>--kind</code></dt>
   <dd>Artifact to write<br/>Possible values: <code>final | raw | salon | manual | metadata</code><br/>Default value: <code>final</code></dd>

   <dt><code>-o/--output</code></dt>
   <dd>Output file. Use "-" to write to stdout</dd>

   <dt><code>-d/--debug</code></dt>
   <dd>Print debug info<br/>Default value: <code>false</code></dd>

   <dt><code>--host</code></dt>
   <dd>Zoo API host to use for all commands (e.g., "api.zoo.dev" or "http://localhost:8888"). Overrides the configured default host</dd>

   <dt><code>-h/--help</code></dt>
   <dd>Print help</dd>
</dl>
