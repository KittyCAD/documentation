## Introduction

This page provides a set of feature spotligth videos for some of Zoo Design Studio's core modeling
features. These videos are intended to be a quick reference both for those users who are unfamiliar
with existing CAD systems, and for those users who are familiar with other CAD software, but new to
Zoo Design Studio.
