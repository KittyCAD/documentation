Use Capture Screenshot to save the current Modeling Area view for design reviews, support requests,
or collaboration without cropping an operating-system screenshot.

## Capture the viewport

1. Open a project and arrange the model in the view you want to share.
2. Click the camera icon in the bottom [Status Bar](/docs/zoo-design-studio/features/workspace/status-bar),
   or open the [Command Bar](/docs/zoo-design-studio/features/workspace/command-bar) and search for
   `Capture screenshot`.
3. In the desktop app, choose where to save the file. In the browser, use the browser's download
   flow.

Zoo Design Studio saves the image as `viewport-screenshot.png`.

## What the screenshot includes

The PNG captures the visible 3D viewport at its current orientation and zoom level. It also includes
the view-orientation gizmo when it is shown. Surrounding interface elements, such as the Toolbar,
panes, Command Bar, and Status Bar, are not included.

Capture Screenshot saves an image of the view; it does not export CAD geometry. To create a file for
another CAD, manufacturing, or visualization tool, use
[Export](/docs/zoo-design-studio/features/data-management/export).

## Troubleshooting

If the camera icon is unavailable, wait for the project and modeling viewport to finish loading. If
the command reports that the viewport is not ready, wait for the model to finish executing and try
again. Zoo Design Studio shows an error notification when it cannot capture or save the image.
