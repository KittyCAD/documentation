Zoo Design Studio is available as a native desktop application on Windows, macOS, and Linux. A web
version is also available for browser-based use.

The desktop applications are the full-featured path. The web version is useful for browser-based
workflows and quick access. It supports multiple files in a single workspace, but full
multi-project workflows are still different from desktop.

In the web version, project files are stored in the browser rather than in a normal project folder
on disk.

## Supported Platforms

- Windows
- macOS
- Linux
- Web browser

## Web Version Caveats

- Multiple files in one workspace are supported. See
  [Web App Multiple Files](/docs/zoo-design-studio/features/system-requirements/web-app-multiple-files).
- Full desktop-style multi-project workflows are not currently supported in the web version.
- Project files are stored in the browser.
