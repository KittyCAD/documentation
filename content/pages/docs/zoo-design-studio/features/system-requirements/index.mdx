Zoo Design Studio is available as a native desktop application on Windows, macOS, and Linux. A web
version is also available for browser-based use.

The desktop applications are the full-featured path. The web version is useful for lighter
workflows and quick access, but it currently does not support multi-project workflows.

In the web version, project files are stored in the browser rather than in a normal project folder
on disk.

## Supported Platforms

- Windows
- macOS
- Linux
- Web browser

## Web Version Caveats

- Multi-project workflows are not currently supported in the web version.
- Project files are stored in the browser.
