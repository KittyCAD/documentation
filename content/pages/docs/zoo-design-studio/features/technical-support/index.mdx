Technical Support covers the main places to ask questions, report bugs, and contact the Zoo team
when you need help using Zoo Design Studio.

For community support, use the [community forum](https://community.zoo.dev/), which runs on
Discourse. It is the best place to search existing answers, ask workflow questions, and post
feature requests.

For faster back-and-forth discussion with the team and other users, use
[Discord](https://discord.gg/JQEpHR7Nt2).

If you need direct email support, paid plans can reach the team through the
[support form](/contact?type=support).
