The Modeling Area is the central 3D scene where sketching, part modeling, selection, and direct
view manipulation happen.

## What You Can Do

- Orbit, pan, and zoom the model.
- Sketch and model directly in the scene.
- Use view controls, gizmos, and unit hints while modeling.
- Select geometry for commands and edits.

## What You Cannot Do

- Reliably run modeling commands when the engine or network is not ready.
- Ignore selection requirements. Some tools require a face, plane, edge, or body first.

## Key Differences in This Area

- Scene edits stay tightly linked to KCL source updates.
- Command availability depends on engine and execution readiness, not only on selection state.
