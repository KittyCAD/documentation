Workspace covers the primary interface areas used to inspect, navigate, and edit the current design
in Zoo Design Studio. This section documents the panels and editors that support modeling workflows
across part modeling, assemblies, and direct KCL editing.

Start with [Toolbar](/docs/zoo-design-studio/features/workspace/toolbar) for quick access to common
modeling tools, or [Feature Tree](/docs/zoo-design-studio/features/workspace/feature-tree) to review ordered
model history and dependencies, [Body Pane](/docs/zoo-design-studio/features/workspace/body-pane) to
inspect the bodies in the current design, and
[Code Editor](/docs/zoo-design-studio/features/workspace/code-editor) to inspect or author KCL
directly. For display overrides such as body color and transparency, see
[Appearance](/docs/zoo-design-studio/features/workspace/appearance). For keyboard-driven command
access, see [Command Bar](/docs/zoo-design-studio/features/workspace/command-bar). For interface and display
preferences, see [Settings](/docs/zoo-design-studio/features/workspace/settings).
