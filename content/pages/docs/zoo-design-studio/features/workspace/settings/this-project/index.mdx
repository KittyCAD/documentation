This Project settings apply only to the currently open project. They are stored in `project.toml`
and override user settings for this project where applicable.

This section follows the same UI structure used in the app:

- [App](/docs/zoo-design-studio/features/workspace/settings/this-project/app)
- [Modeling](/docs/zoo-design-studio/features/workspace/settings/this-project/modeling)
- [Text Editor](/docs/zoo-design-studio/features/workspace/settings/this-project/text-editor)
- [Command Bar](/docs/zoo-design-studio/features/workspace/settings/this-project/command-bar)
- [Meta](/docs/zoo-design-studio/features/workspace/settings/this-project/meta)
- [Resets](/docs/zoo-design-studio/features/workspace/settings/this-project/resets)
- [About Design Studio](/docs/zoo-design-studio/features/workspace/settings/this-project/about-design-studio)

Use this tab when a project needs different defaults from your global user settings.
