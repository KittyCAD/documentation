The `Resets` section contains recovery actions for onboarding, saved settings, and workspace layout.

## Onboarding

### Replay Onboarding

Replay the onboarding process.

## Reset Settings

Restore settings to their default values.

Your settings are saved in a file in the app data folder for your operating system.

On desktop, the reset area also includes `Show in folder`. The reset action for this tab is `Reset
user-level settings`.

## Layout

Reset to the default layout.
