The `Modeling` section contains user-level defaults for units, viewport display, navigation, grid
behavior, and sketching preferences.

## Included Settings

### Default Unit

Set the default length unit setting value to give any new files.

### Highlight Edges

Whether to highlight edges of 3D objects.

### Back-Face Color

Default backface color for surfaces. For details, see
[Back-Face Color for Surfaces](/docs/zoo-design-studio/features/workspace/settings/user-settings/modeling/back-face-color-for-surfaces).

### Enable Touch Controls

Enable touch events to navigate the 3D scene. When enabled, orbit with one-finger drag, pan with
two-finger drag, and zoom with pinch.

### Use Sketch Solve Mode

Default to the experimental solver-based sketch mode for all new sketches.

### Camera Projection

Projection method applied to the 3D view, perspective or orthographic.

### Camera Orbit

What methodology to use for orbiting the camera.

### Gizmo Type

Which type of orientation gizmo to use.

### Show Scale Grid

Whether to show a scale grid in the 3D modeling view.

### Fixed Size Grid

When enabled, the grid will use a fixed size based on your selected units rather than automatically
scaling with zoom level.

### Major Grid Spacing

The space between major grid lines, specified in the current unit.

### Minor Grids Per Major

Number of minor grid lines per major grid line.

### Snap to Grid

Snap the cursor to the unit grid when drawing lines, arcs, and other segment-based tools.

### Snaps Per Minor

Number of snaps between minor grid lines. `1` means snapping to every minor grid line.
