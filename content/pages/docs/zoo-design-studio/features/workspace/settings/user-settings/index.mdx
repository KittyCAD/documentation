User Settings are the main user-level preferences shown in the settings UI.

They apply globally to your account and all projects, and are stored in `user.toml`.

This section is organized in the same order used in the app:

- [App](/docs/zoo-design-studio/features/workspace/settings/user-settings/app)
- [Modeling](/docs/zoo-design-studio/features/workspace/settings/user-settings/modeling)
- [Text Editor](/docs/zoo-design-studio/features/workspace/settings/user-settings/text-editor)
- [Projects](/docs/zoo-design-studio/features/workspace/settings/user-settings/projects)
- [Command Bar](/docs/zoo-design-studio/features/workspace/settings/user-settings/command-bar)
- [Resets](/docs/zoo-design-studio/features/workspace/settings/user-settings/resets)
- [About Design Studio](/docs/zoo-design-studio/features/workspace/settings/user-settings/about)

These settings are primarily user-level preferences, though some controls are desktop-only or vary
by platform and build.
