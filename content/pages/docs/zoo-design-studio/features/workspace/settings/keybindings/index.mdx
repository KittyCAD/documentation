The `Keybindings` tab shows keyboard shortcuts for actions throughout Zoo Design Studio. Shortcuts
are grouped by workflow and are platform-aware.

## Sketching

- `Enter Sketch Mode`: `S`
- `Unequip Sketch Tool`: `Escape`
- `Exit Sketch Mode`: `Escape`
- `Toggle Line Tool`: `L`
- `Toggle Rectangle Tool`: `R`
- `Toggle Arc Tool`: `A`

## Modeling

- `Extrude`: `E`
- `Center on selection`: `Ctrl+Alt+C` (`Cmd+Option+C` on macOS)
- `Reset camera position`: `Ctrl+Alt+X` (`Cmd+Option+X` on macOS)

## Command Palette

- `Toggle Command Palette`: `Ctrl+K` (`Cmd+K` on macOS)

## Settings

- `Toggle Settings`: `Ctrl+,` (`Cmd+,` on macOS)
- `Settings Search`: `Ctrl+.`

## Panes

- `Toggle Code Pane`: `Shift+C`
- `Toggle Variables Pane`: `Shift+V`
- `Toggle Logs Pane`: `Shift+L`
- `Toggle Errors Pane`: `Shift+E`

## Code Editor

- `Format Code`: `Shift+Alt+F`

## File Tree

- `Rename File/Folder`: `Enter`
- `Delete File/Folder`: `Ctrl+Backspace` (`Cmd+Backspace` on macOS)

## Miscellaneous

- `Undo`: `Ctrl+Z` (`Cmd+Z` on macOS)
- `Redo`: `Ctrl+Shift+Z` (`Cmd+Shift+Z` on macOS)
