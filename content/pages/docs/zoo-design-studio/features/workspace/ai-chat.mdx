AI Chat houses the Zookeeper conversation pane inside Zoo Design Studio. Use it to ask questions
about the current project, request geometry creation or edits, and work through modeling tasks in
natural language.

## Core Concepts

### Prompt

A Prompt is the message or instruction you send to Zookeeper. Prompts can be questions, requests to
generate or edit geometry, or any other task you want the agent to handle.

### Response

A Response is the message Zookeeper sends back after processing a prompt. Responses can include
explanations, follow-up questions, or actions taken in the project, and may suggest next steps.

### Tool

A Tool is a specific capability Zookeeper can use inside Design Studio, such as creating geometry,
editing a feature, or reading project state.

### Cancel

Cancel stops the current Zookeeper operation before it finishes. Use it when you want to interrupt
the current request and change direction.

### Reasoning

Reasoning is the internal process Zookeeper uses to decide what to do. It is not always shown
directly, but it shapes the response and the actions taken in the project.
