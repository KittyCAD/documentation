The Toolbar is the main strip of buttons at the top of the modeling app, providing quick access to the most important modeling tools like [Extrude](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude), [Revolve](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/revolve), [Loft](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/loft), [Sweep](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/sweep), and more.

## What the Toolbar Does

The Toolbar lets you:
- Start a new sketch or edit an existing one
- Turn sketches into 3D shapes (for example, using [Extrude](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude) or [Revolve](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/revolve))
- Add features like [fillets](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/fillet) (rounded edges) or [chamfers](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/chamfer) (angled edges)
- Quickly switch between different modeling actions
- Access sketch tools and constraints while in sketch mode

Each button in the Toolbar has an icon and a short label. When you hover over a button, you'll see a tooltip that explains what the tool does. Some buttons also have keyboard shortcuts, so you can work even faster.

## Key Benefits

### Always Visible
The Toolbar is always visible at the top of the screen, so you don't have to search through menus to find the tools you need.

### Context-Aware
Buttons are enabled or disabled depending on what you're doing. For example, you can only [Extrude](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude) when you have a sketch selected.

The Toolbar also changes with modeling state. Sketch mode exposes sketch geometry, constraints, and
sketch editing tools, while 3D modeling exposes solid, surface, boolean, transform, and
construction workflows.

### Helpful Tips
Tooltips give you a quick explanation of each tool, and sometimes link to more detailed documentation or examples.

### Keyboard Shortcuts
Many tools have keyboard shortcuts (like E for [Extrude](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude), R for [Revolve](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/revolve)) so you can keep your hands on the keyboard and work faster.

## Common Toolbar Tools

Most modeling workflows start with [**Sketch**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/sketching), where you draw a 2D shape on a selected plane or face. Think of this as laying out your design on paper before bringing it into 3D. Once you have your sketch, the real magic begins.

The most common next step is [**Extrude**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude), which pulls your sketch profile perpendicular to the sketch plane, creating solid 3D geometry. It's perfect for simple shapes like boxes, brackets, or any feature that extends straight out.

For round parts like bottles, shafts, or anything with circular symmetry, reach for [**Revolve**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/revolve). This tool spins your sketch profile around an axis, creating turned geometry in seconds—think of a potter's wheel transforming a simple profile into a complete vessel.

When you need to blend between different shapes, [**Loft**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/loft) smoothly transitions between two or more sketch profiles. It's ideal for organic shapes or tapered transitions where the cross-section changes along the length. [**Sweep**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/sweep) takes a similar approach but moves your sketch profile along a path, perfect for pipes, handrails, or any geometry that follows a curve.

After creating your basic shapes, you'll often want to soften sharp edges. [**Fillet**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/fillet) rounds off edges with smooth tangent surfaces, reducing stress concentrations and making parts more comfortable to handle. If you prefer a crisper look or need easier machining, [**Chamfer**](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/chamfer) creates beveled transitions instead of curved rounds.

## Using the Toolbar

1. **Select geometry** - Click on a sketch, face, or edge depending on what tool you want to use
2. **Click a tool button** - The Toolbar will show which tools are available based on your selection
3. **Enter parameters** - A dialog or panel appears where you specify dimensions and options
4. **Confirm** - Apply the operation to your model

If you're ever unsure what a button does, hover over it to see a description and helpful links to detailed documentation.

## Availability and State

Toolbar commands depend on more than selection alone:

- Some tools are unavailable while the engine connection is down or model execution is in progress.
- Some commands are intentionally marked `experimental`, `unavailable`, or `kcl-only`.
- Some in-progress behaviors are present but not yet production-complete.

## Toolbar vs Command Bar

The Toolbar provides visual, mouse-driven access to frequently-used modeling operations. For keyboard-driven workflows or less common commands, use the [Command Bar](/docs/zoo-design-studio/features/workspace/command-bar) instead.
