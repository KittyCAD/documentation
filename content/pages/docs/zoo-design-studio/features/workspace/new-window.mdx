New Window opens another Zoo Design Studio window so the current work can keep running while you
start or inspect a separate session. Use it when you want to keep a model, AI run, or project view
available while opening another workspace.

## Command

Run the New Window command from the app menu or the Command Bar. The new window opens as a separate
workspace shell.

## Current Work

The existing window stays open. Any running modeling task, Zookeeper conversation, or project
inspection can continue while the new window starts.

## New Session

Use the new window to open another project, start a fresh design, or compare work without replacing
the current view.

## Platform Notes

New Window is a desktop workflow. Browser-based sessions already run in browser tabs or windows, so
use your browser's window controls there.

## Are you interested in code?

New Window is a workspace command, not a KCL modeling function. It does not change the model code by
itself.

For related workspace controls, see [Top Bar](/docs/zoo-design-studio/features/workspace/top-bar)
and [Command Bar](/docs/zoo-design-studio/features/workspace/command-bar).
