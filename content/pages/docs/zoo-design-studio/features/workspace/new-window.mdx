New Window opens another Zoo Design Studio desktop window without closing the current one.

## Use It For

- Opening another project.
- Starting a fresh workspace.
- Keeping a model, project view, or Zookeeper conversation visible while checking something else.

## How To Open

Run New Window from the app menu or the Command Bar.

## Notes

New Window is a desktop workflow. In the web app, use normal browser tabs or windows instead.
