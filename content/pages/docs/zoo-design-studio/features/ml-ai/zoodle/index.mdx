Zoodle lets you draw directly on the model viewport and send that visual context to Zookeeper. Use
it when a marked-up region, edge, or face is clearer than a text-only instruction.

## Drawing Area

Open Zoodle from the Zookeeper prompt controls. Draw close to the geometry you want Zookeeper to
notice.

## Prompt

After sending the marked-up image back to the composer, add a short prompt that explains the edit:
"move this hole here", "round this edge", or "make this wall thicker".

## Model Context

Zookeeper receives the current model context with the drawing. Rotate or zoom the model first when a
different view makes the target easier to identify.

## Iteration

Use another Zoodle markup when the first result needs refinement.

For the full assistant workflow, see [Zookeeper](/docs/zoo-design-studio/zookeeper) and
[AI Chat](/docs/zoo-design-studio/features/workspace/ai-chat).
