title: Assemblies
excerpt: Assembly workflows in Zoo Design Studio.
sidebarPosition: 5
---

Assemblies define how separate parts are positioned and oriented within a single design context.
Current assembly workflows focus on inserting single-body parts into the main assembly. Nested
subassemblies and broader multi-body insertion workflows are not yet part of the supported assembly
flow. For the current in-app workflow, start with
[Insert](/docs/zoo-design-studio/features/3d-design/assemblies/insert).
