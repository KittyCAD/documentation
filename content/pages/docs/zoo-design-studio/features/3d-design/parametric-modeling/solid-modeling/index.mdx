Solid modeling tools turn sketches and bodies into manufacturable geometry. This section groups solid
features into body creation, body modification, and boolean operations so feature intent stays easy
to scan.

  alt="Sectioned solid model shown in Zoo Design Studio"
  aspect="1846 / 2422"
  maxHeightPx={640}
/>

Start with [Create Solids](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids)
for shape-building features such as
[Extrude](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/extrude),
[Sweep](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/sweep),
[Loft](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/loft),
and [Revolve](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/create-solids/revolve).
Use [Modify Solids](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids)
for [Fillet](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/fillet),
[Chamfer](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/chamfer),
[Shell](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/shell),
and [Hole](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/modify-solids/hole).
For overlap-based body editing, see
[Boolean Operations](/docs/zoo-design-studio/features/3d-design/parametric-modeling/solid-modeling/boolean-operations).
