Use Extrude to pull a 2D sketch into 3D geometry. It is one of the fastest ways to create base
bodies, bosses, pockets, and symmetric feature additions or removals.

## Extrude Symmetric

Extrude Symmetric grows a feature equally in opposite directions from its sketch plane. Use it when
the feature should stay centered on its original reference.

## Extrude to Faces

Extrude to Faces uses existing model faces as end conditions instead of a fixed depth. That keeps
the feature tied to surrounding geometry as the design evolves.

### Extrude to Standard Sweep Faces

This workflow extends extrusions until they meet standard sweep faces, which is useful when the
target boundary is driven by an existing path-based surface or body.
