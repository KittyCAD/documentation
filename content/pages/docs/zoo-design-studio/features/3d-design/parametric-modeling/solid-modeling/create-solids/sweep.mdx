Use Sweep to carry a 2D sketch profile along a 3D path, generating complex geometry that follows curves and arbitrary trajectories. Sweep is ideal for creating pipes, rails, cable routes, handrails, and other features that follow non-linear paths.

## Sketches
Select a closed sketch region as the sweep profile. A closed sketch encloses an area, and Zoo Design Studio exposes that filled area as a selectable region. For a solid sweep, select the region as the profile, not the individual sketch edges.

## Relative To
Controls how the profile orients itself as it follows the path.

## Start Tag and End Tag
Tag the start face (where the profile begins) or end face (where it finishes) to reference them later. This is mainly used for referencing faces from other objects or features, especially in code or scripts. In the usual point-and-click UI flow, you don't need to set these tags—it's mostly for advanced users who want precise control or automation.

## Body Type
Choose whether to create a <b>solid body</b> (watertight volume) or a <b>surface body</b> (shell with no thickness). Most mechanical parts use <b>solid body</b>.

## Are you interested in code?

Zoo Design Studio writes KCL behind the scenes. Here’s an example of a solid sweep in KCL:

```kcl
profileSketch = sketch(on = XZ) {
    line4 = line(start = [var 15mm, var 30mm], end = [var -15mm, var 30mm])
    line5 = line(start = [var -15mm, var 30mm], end = [var -15mm, var -30mm])
    coincident([line4.end, line5.start])
    line6 = line(start = [var -15mm, var -30mm], end = [var 15mm, var -30mm])
    coincident([line5.end, line6.start])
    line8 = line(start = [var 15mm, var -30mm], end = [var 15mm, var 30mm])
    coincident([line8.end, line4.start])
    horizontal(line6)
    horizontal(line4)
    vertical(line5)
    verticalDistance([line5.end, line5.start]) == 60
    horizontalDistance([line6.start, line6.end]) == 30
    coincident([line6.end, line8.start])
    vertical(line8)
    horizontalDistance([ORIGIN, line8.end]) == 15
    verticalDistance([ORIGIN, line8.end]) == 30
}
guideSketch = sketch(on = XY) {
    line1 = line(start = [var 0mm, var 0mm], end = [var 0mm, var 80mm])
    coincident([line1.start, ORIGIN])
    vertical([line1.end, ORIGIN])
    line2 = line(start = [var 40mm, var 120mm], end = [var 70mm, var 120mm])
    arc1 = arc(start = [var 40mm, var 120mm], end = [var 0mm, var 80mm], center = [var 40mm, var 80mm])
    coincident([arc1.end, line1.end])
    coincident([arc1.start, arc1.start])
    coincident([arc1.start, line2.start])
    tangent([line1, arc1])
    tangent([line2, arc1])
    horizontal(line2)
    verticalDistance([line1.start, line1.end]) == 80
    horizontalDistance([line2.start, line2.end]) == 30
    radius(arc1) == 40
}
profileRegion = region(point = [0mm, 0mm], sketch = profileSketch)
sweepSolid = sweep(
    profileRegion,
    path = [
        guideSketch.line1,
        guideSketch.arc1,
        guideSketch.line2
    ],
)
```
