Mirror reflects selected 3D geometry across a reference plane, edge, axis, or sketch segment. Use
Mirror for symmetric parts, mirrored design variants, and creating paired surface or solid bodies
without manually rebuilding the second side.

## Options

### Objects
Select the surface or solid bodies you want to mirror. Multiple bodies can be mirrored together.

### Across
Choose the geometry that defines the mirror reference. This can be a default plane, a construction
plane, an edge, an axis, or a sketch segment.

Behind the scenes, Mirror maps to KCL's
[`mirror3d`](/docs/kcl-std/functions/std-transform-mirror3d) function.
