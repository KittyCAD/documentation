---
title: Scale
excerpt: Scale geometry in Zoo Design Studio.
sidebarPosition: 2
---

Scale changes geometry size along one or more axes, which is useful for proportional edits,
controlled stretch operations, and fast concept variations.

Use Scale when geometry needs to be resized in one direction, across a plane, or uniformly in all
directions. The same feature supports these different scaling modes, even though the controls and
parameters may vary by modeling context.
