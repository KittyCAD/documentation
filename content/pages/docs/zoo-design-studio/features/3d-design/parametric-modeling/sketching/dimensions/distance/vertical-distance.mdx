---
title: Vertical Distance
excerpt: Define vertical spacing between sketch entities in Zoo Design Studio.
sidebarPosition: 0
---

Vertical Distance measures and controls spacing along the sketch's vertical direction. Use it when
the design intent depends on height rather than direct point-to-point distance.

Select a line when the vertical span of that segment should be driven, or select two points or
references to control their vertical spacing. Double-click an existing vertical distance label to
edit its value.
