Sketch tools modify existing sketch geometry without rebuilding the entire profile. Use them to clean
up curves, remove unwanted overlaps, and refine profile boundaries before creating 3D features.

## Trim

Trim removes or shortens sketch segments back to the nearest intersecting geometry. Use it to clean
up overbuilt profiles and refine closed regions before creating downstream features.
