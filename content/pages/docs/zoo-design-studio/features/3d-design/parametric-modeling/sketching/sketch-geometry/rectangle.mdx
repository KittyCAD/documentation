Use Rectangle to create four-segment rectangular profiles quickly. Rectangles are commonly used for
pads, pockets, slots, and base profiles that will later drive extrudes, cuts, or patterns.

## Corner Rectangle

Corner Rectangle defines the profile from one corner to the opposite corner. Use it when one corner
of the rectangle is already anchored to existing sketch geometry, an origin, or a construction
reference.

## Center Rectangle

Center Rectangle defines the profile from its center point outward. Use it when the rectangle needs
to stay centered on a reference point, centerline, or axis of symmetry.

## Angled Rectangle

Angled Rectangle creates a rectangular profile that is rotated relative to the sketch axes. Use it
when the profile must follow a non-orthogonal edge, reference direction, or layout angle.
