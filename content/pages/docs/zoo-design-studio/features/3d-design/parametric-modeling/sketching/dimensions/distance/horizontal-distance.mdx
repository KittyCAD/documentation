---
title: Horizontal Distance
excerpt: Define horizontal spacing between sketch entities in Zoo Design Studio.
sidebarPosition: 1
---

Horizontal Distance measures and controls spacing along the sketch's horizontal direction. Use it
when the design intent depends on lateral offset instead of direct point-to-point distance.

Select a line when the horizontal span of that segment should be driven, or select two points or
references to control their horizontal spacing. Double-click an existing horizontal distance label to
edit its value.
