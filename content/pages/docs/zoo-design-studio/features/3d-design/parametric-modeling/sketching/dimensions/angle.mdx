---
title: Angle
excerpt: Control the angle between sketch lines with an angle dimension in Zoo Design Studio.
sidebarPosition: 1
---

The Angle dimension controls the included angle between two sketch lines. Use it when the design
intent depends on an explicit angular value instead of only horizontal, vertical, or perpendicular
alignment.

Double-click an existing angle dimension label to edit the angle value after it has been placed.
